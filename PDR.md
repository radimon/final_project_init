# Preliminary Design Review (PDR)

A. Attention Getter（動機 / 引題） 
隨著智慧家居與極簡主義設計的快速發展，使用者對於室內設備的需求已不僅
限於功能性，更重視整體空間的美觀與整潔性。然而，傳統機械式開關通常需
要外露於牆面或家具表面，不僅破壞視覺一致性，也容易因灰塵堆積或長期使
用造成機械損耗。因此，如何在不影響外觀的情況下實現直覺且自然的人機互
動，成為值得探討的研究議題。 
B. But（挑戰 / 問題） 
現有市售觸控開關多依賴專用面板或需直接暴露於使用者可觸及的位置，導致
安裝彈性受限。此外，部分解決方案如影像辨識或感測器系統，雖具備非接觸
操作能力，但往往成本較高且系統複雜。如何在「低成本」、「高隱蔽性」與
「穩定性」之間取得平衡，仍是一項關鍵挑戰。 
C. Cure（解法 / 核心貢獻） 
為了解決上述問題，本研究提出一種基於 ESP32 電容感測技術的隱形觸控開關
系統。該系統利用電容變化偵測人體接近行為，並可穿透非金屬材料（如木板
或壓克力），在不改變既有家具結構的情況下實現隱藏式操作介面，提供更自
然且直覺的互動體驗。 
D. Development（系統設計 / 方法） 
本系統以 ESP32 內建的 touch sensor 為核心，透過導電鋁箔作為感測電極，
偵測人體手指靠近時產生的微小電容變化。為提升穩定性，系統採用多次取樣
平均（moving average）與閾值判斷機制，以降低環境雜訊與靜電干擾的影
響。此外，系統整合 Wi-Fi 模組，透過 LINE Notify 實現遠端通知與裝置控
制，並支援不同觸控模式（如短觸與長按）以提升功能擴展性。 
E. Experiments（實驗設計） 
為評估系統性能，本研究針對不同材質（木板、壓克力）與不同厚度（3mm、
5mm、10mm）進行穿透測試。實驗過程中記錄感測原始值（raw value）與觸發
閾值之關係，並分析在不同條件下的觸發穩定性與誤觸情形。同時，本研究亦
測量觸發反應時間與遠端通知延遲，以評估系統實用性。 
 
 
 

F. Findings（結果 / 貢獻） 
實驗結果顯示，本系統能穩定穿透厚度 10mm 以下之非金屬材料，且在適當閾
值設定下，誤觸率可控制於 2% 以下。此外，系統整體延遲低於 1 秒，具備即
時回應能力。由於本設計僅需低成本材料（如鋁箔與開發板），在實務應用上
具備高度可行性與擴展潛力，適用於智慧家居與隱形介面相關應用場景。 
 
 
 
2. Related Work / NotebookLM 
在現代智慧家居與人機互動領域中，已存在多種實現控制與互動的方式。最常
見的方式為傳統機械式開關，其優點為結構簡單且穩定，但缺點是外觀明顯，
無法融入極簡設計風格。 
近年來，觸控式面板逐漸普及，透過電容感測實現操作，例如智慧燈控開關。
然而，這類產品通常需將觸控區域直接暴露於表面，限制了其隱藏式應用的可
能性。 
此外，一些系統採用影像辨識或手勢辨識技術，利用攝影機進行非接觸控制，
但這類方法通常需要較高運算資源，成本較高且系統複雜度較大。 
在開發工具方面，本研究採用 ESP32 作為核心控制器，並利用其內建電容感測
功能（Touch Sensor）進行輸入偵測。同時，透過 Arduino IDE 進行程式開
發，並結合 LINE Notify 作為即時通知工具，實現簡單且低成本的物聯網應
用。 
綜合上述，本研究選擇以「低成本硬體 + 內建感測能力 + 即時通訊工具」作
為設計基礎，並進一步發展可穿透材質的隱形互動方式，以彌補現有方法在隱
蔽性與安裝彈性上的不足。 
 
 
 
 

 
3. Proposed Design（系統規格） 
本系統設計為一套隱形觸控開關，主要由硬體模組與軟體控制兩部分構成。 
在硬體方面，系統使用 ESP32 開發板作為核心控制單元，並利用其內建的電容
觸控腳位（Touch GPIO）進行訊號感測。感測電極採用鋁箔紙製作，並貼附於
桌面或木板下方，透過非金屬材質（如木板或壓克力）與使用者手指形成電容
變化。此外，系統透過杜邦線連接電極與 ESP32，以完成基本電路架構。 
在軟體方面，系統使用 Arduino IDE 進行開發，透過 touchRead() 函式讀取
感測值。為提升穩定性，系統採用多次取樣平均（moving average）方式進行
數據平滑處理，並設定觸發閾值（threshold）來判斷是否為有效觸控。此外，
系統支援不同操作模式，例如短觸（開關燈）與長按（啟動警示模式），以提
升互動彈性。 
在通訊方面，ESP32 透過 Wi-Fi 連接網路，並整合 LINE Notify API，當偵測
到觸控事件時，系統會即時發送通知至使用者手機，實現遠端監控與控制功
能。 
整體而言，本系統具備低成本、易於安裝及高隱蔽性等特點，且不需改變既有
家具結構，即可實現自然直覺的人機互動。
