---
description: Create change numbering rule
---
When executing a "create change" process or creating a new proposal, you must apply the following numbering rule:
- The numbering for new changes must start from `01-`.
- Subsequent changes should increment this number sequentially (e.g., `02-`, `03-`, etc.).
- Ensure that the numbering format includes the two-digit zero-padded number followed by a hyphen before the rest of the change name or description.
